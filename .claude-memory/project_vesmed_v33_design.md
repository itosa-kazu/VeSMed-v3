## 目標架構 v3.3（感覚系統から逆設計）

### データ層（不変）
- 変量901個（≈嗅覚受体）
- 辺8201条 D→E（≈結合亲和力）
- CPT（≈結合強度）
- D→D辺: 新増（疾患間因果鏈）

### 推論引擎: 三層単一回路

```
第1層: 原始信号（Noisy-OR組合規則 — 保留）
  → 各疾患×各症状の亲和力計算
  → 嗅覚: 受体の原始激活値

第2層: D→D因果伝播（v3.2 新増）
  → 活性疾患が下流疾患をブースト
  → 嗅覚: 対応なし（VeSMed固有）

第3層: Explaining away + 状態記憶（v3.3 核心）
  ├ 快照内: 競争的credit assignment（侧抑制）
  └ 快照間: 記憶による適応（顆粒細胞）
  同一回路、同一操作（既説明の抑制）
```

### 削除
- IDF重み（explaining awayが自然代替）
- 独立打分（競争的credit assignmentに置換）

### 涌現する副産物
- unexplained residual検出器（違和感メカニズム）
- 出力形式: 排名表→疾患組合+各疾患の認領症状

### Noisy-ORの新しい位置づけ
- Noisy-OR組合規則（局部）: **保留** — 多因独立産生効果の生成モデル
- Noisy-OR独立打分（全局）: **削除** — explaining awayに置換
- 「受体の原始信号計算」に降格。嗅球処理はexplaining awayが担う
