## Scope外（非発熱主訴、将来の拡張時に検討）
- **AMI/ACS** — 胸痛主訴。国試最重要だがVeSMed発熱domainでは対象外
- **大動脈解離** — 胸背痛主訴。同上
- **安定狭心症** — 労作性胸痛
- **慢性MR** — 慢性経過、D216(急性MR)で一部カバー
- **慢性AR** — 慢性経過、D215(急性AR)で一部カバー
- **WPW症候群** — D356(SVT)に含まれる
- **QT延長症候群** — 薬剤性含む、D357(VT)と重複
- **Eisenmenger症候群** — 先天性心疾患合併症
- **肺性心(cor pulmonale)** — D214(PAH)で部分カバー
- **心臓サルコイドーシス** — D62(全身サルコイドーシス)に包含
- **虚血性心筋症** — AMIのsequela、D120(ADHF)でカバー
- **頻脈性心筋症** — D356(SVT)のsequela

**How to apply:** 心臓系は18疾患で主要疾患を網羅済み。次のアクションはD215/D356の案例追加(find-cases)が最優先。
