辺追加後にTop-1が上がってもTop-3が下がった場合（またはその逆）、**「正常範囲」と流さず必ず調査**すること。

**Why:** ユーザーの指摘：「框架と理論が正しければ、正しい辺を追加すれば全指標が改善するはず。一方が下がるのは異常信号」。Top-3 581→580(-1)を「微減は正常」と流したが、本来は調査が必要だった。

**根本原理:** Noisy-OR BNでは、正しい因果辺の追加は：
1. 期待疾患の尤度を上げる（→rank改善）
2. 他の疾患の尤度は変わらない（辺は片方向）
したがって理論上、正しい辺追加で他案例のrankが悪化することはない。悪化した場合は：
- その辺のCPT値が不適切（他疾患との競合を変えた）
- 辺自体が過拟合（臨床的に不正確）
- 案例のevidence入力ミス

**How to apply:**
1. 辺追加後の回帰テストで**全指標を確認**（Top-1, Top-3, FATAL）
2. いずれかが**1件でも悪化**したら、悪化した案例を特定して原因調査
3. 原因が辺のCPT値なら修正、辺自体が不適切なら撤回
4. 「微減」「正常範囲」という表現で流さない
