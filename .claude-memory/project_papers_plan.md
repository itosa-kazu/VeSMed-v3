## 论文一：LLM生成BN的7轮审查protocol + NO EDGE问题
- **核心命题：** LLM生成BN的最大威胁不是CPT参数错误，而是边的结构性缺失，导致"自信的误诊"（H<2却rank>3）
- **已有弹药：**
  - 5个confirmed自信误诊案例，全部根因=NO EDGE
  - 7轮LLM审查protocol（让LLM自己检查自己生成的BN）
  - 102疾患、984辺的完整网络作为showcase
- **未完成：**
  - HEPAR II（肝病BN经典benchmark）用LLM重新生成，和原版对比
  - 盲检50例以上（目前36 in-scope real case）
  - 对比实验：LLM一次生成 vs 7轮审查后的辺遗漏率
- **就绪度：85%**

## 论文二：反証推奨 — 波普尔証伪主義の初の計算実装
- **核心命题：** 従来のCDSSは確認方向(エントロピー減少)の検査を推奨するが、これは確認バイアスを助長する。反証方向(エントロピー増大)の検査推奨により、誤診時は自己修正、正診時は補強する
- **方法：** max_s(H_after(v=s)) — 各検査の最もエントロピーを増大させる状態で評価
- **已有弹药：**
  - PMC419768骨髄腫FUO: 反証推奨5ステップでTop-1到達 vs 通常推奨18ステップでも到達不能
  - R01 Still病: 反証推奨でTop-1が30%→99.9%に強化（正診時は補強される実証）
  - 実装済み(bn_inference.py + app.py + UI)
  - 先行研究調査済み: 完全新規（MSのMAI-DxO Dr.Challengerが最も近いがLLM依存で非数理的）
- **未完成：**
  - 多症例での検証（骨髄腫以外でも有効か）
  - 反証→確認の切替戦略の検討
  - 数学的な理論的保証（なぜ反証が確認バイアスを防ぐか）
- **目標期刊:** JAMIA, J Biomed Inform, AI in Medicine, AAAI Health Track
- **就绪度：90%** ← 実装・先行研究・実証データ全て揃い

## 论文三：エントロピー減少曲線 → 「次にどの検査をすべきか」
- **核心命题：** 每加一个所見，posteriorのエントロピーH下降。H下降速度和方向は疾患の"指纹"
- **已有弹药：**
  - IE→SAHの6步scenario：H 4.9→2.1→2.3→1.4→0.4
  - next_best_test推奨機能実装済み
  - 反証推奨との比較データ
- **就绪度：70%**（论文二に統合する可能性あり）
