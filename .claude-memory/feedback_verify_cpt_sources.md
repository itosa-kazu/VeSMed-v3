CPT数値を設定する際、研究agentが引用した文献の百分比を**必ず原文fetchで検証**すること。

## ルール
1. 研究agentがPMID/PMCと百分比を報告しても、**そのまま使わない**
2. 必ずWebFetchで原文を取得し、該当する百分比が本文/Table/Figureに実在するか確認する
3. 原文で確認できた数値のみCPTに使用する
4. 確認できない場合は、別の文献を探すか、教科書の記述で代替する

**Why:** 研究agentのweb searchは要約や二次引用から数値を拾うことがあり、原文の実際の記述と異なる場合がある。CPTは診断精度に直結するため、根拠の正確性は最重要。2026-03-19に7辺を追加した際、agent報告の百分比を直接検証せずにCPT設定した。

**How to apply:** 辺追加/CPT修正のワークフローで、文献調査後に必ず「原文fetch検証」ステップを入れる。特にagentが報告した数値は必ずダブルチェック。
