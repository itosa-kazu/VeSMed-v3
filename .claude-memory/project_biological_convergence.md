## 核心洞察

万有医学因果図は人為発明ではなく、**生命系統の普遍架構の再発見**。

全層に共通する4要素：
1. **組合編码**（少量組分→海量模式）
2. **侧抑制/競争選択**（強者先占、弱者分余）
3. **多層伝播**（逐層処理、非一步到位）
4. **反馈調控**（自頂向下修正）

框架のrobustness（全対抗攻撃通過）の説明：convergent evolution = 生命が異なる尺度で反復検証済みの解。

**Why:** 嗅覚類比からの拡張。因果図架構が「正しく感じる」理由は、生物が4億年かけて同じ解に収束しているから。

**How to apply:** 生物系統から推論引擎の改良ヒントを得る。特に侧抑制(explaining away)と表観遺伝(context-dependent CPT)。冗余通路(複数検査)は偽陰性問題の臨床的解決法。
