## Netflix反論（致命的）

**Netflix とVeSMedの目標は正反対。**

- Netflix: 似た映画を推薦 → 類似性で補完 = 正しい
- VeSMed: 似た疾患を鑑別 → 類似性で補完 = **鑑別情報を破壊**

例: 肺炎と肺結核はほぼ同じCPTプロファイル（咳嗽、発熱、浸潤影）。
NMFは「肺炎≈肺結核」と学習し、欠損CPTを似た値で埋める。
しかし診断に必要なのは**差異**（盗汗は結核にあり肺炎にない）。
NMFは差異を系統的に補完できない — むしろ差異を消す方向に働く。

**Paper 1のconfident misdiagnosis問題と同構:**
- 辺欠失 = 類似疾患間の微細差異が未記録
- Netflix式補完 = 差異ではなく共通性を復元
- → 最も必要な情報を系統的に補完不能

## 結論

1. **形状向量は存在する** — CPT矩陣は低秩（有効次元≈74）、因子は病理生理に対応
2. **しかし補完には使えない** — 診断=鑑別、鑑別情報=類似疾患間の例外、NMFは例外を消す
3. **嗅覚の物理免費計算にはならない** — 物理は差異も共通性も等しく精確。統計的補完は共通性偏重
4. **大清洗（人手+文献）は依然不可避** — 特に鑑別に重要な辺ほど自動補完不能、人手確認が必須

**Why:** 嗅覚形状互補仮説の実証実験+理論的限界の発見。
**How to apply:** 低秩構造の知見はQCツール（異常CPT検出）に活用可能。補完には使わない。
