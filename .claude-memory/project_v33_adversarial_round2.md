## 各攻撃の詳細

### 攻撃1: EA初期化依存性（⭐⭐⭐⭐→⭐ 解決済）
**問題**: EAは逐次的にcredit assignmentを行う。最初にどの疾患が症状を「認領」するかで、後続の疾患の評価が変わる。処理順序が結果を左右する。
**嗅覚との対比**: 嗅覚の侧抑制は**並列**（全神経元が同時に競争）。VeSMedのEAが**逐次**なら、嗅覚と非同構。
**解法**: **Jacobi同期更新**。全疾患が「前轮の快照」を基に新claim値を計算→全部計算完了後に同期更新。数学的に並列競争と等価。収束保証はsoft claiming(∈(0,1))による圧縮映射+Banach不動点定理。
**先行事例**: QMR-DT(Shwe 1991)が同様のBN推論を試みたが当時は計算不可能。現在の430疾患規模では実用的。

### 攻撃2: 記憶誤差鎖死（⭐⭐⭐⭐⭐）最危険 — 議論中
**問題**: 快照1で疾患Aが症状Xを誤って認領→記憶に格納→快照2で症状Xは「既解釈」として無視→真の原因Bが永遠に発見されない。誤差が自己強化するpositive feedback loop。
**生物対照**: 免疫のautoimmune disease = 同じ失敗モード（自己を敵と誤認→記憶固着→慢性攻撃）。嗅覚のcross-adaptation = 軽度版（気味A適応→気味B知覚歪曲）。
**嗅覚が軽度で済む理由**: (1)適応が自動衰減 (2)soft抑制(圧低≠消灭) (3)新強信号が覆蓋可能
**核心洞察**: 嗅覚記憶は**信号級**(原始激活パターンを圧低)、VeSMed記憶が**解釈級**(「症状Xは疾患Aのもの」と記憶)だと鎖死する。信号級記憶にすべき。
**三安全弁設計(2026-04-06)**:
1. **衰減(Decay)**: 記憶weight w *= decay_rate(≈0.8)/快照。3-5快照で誤差が自然消退
2. **Soft抑制**: 認領は確率加重(0,1)、hard 0/1判定なし。被認領済み症状も競争に参加可能
3. **強信号覆蓋(Override)**: 新LRが閾値超→旧認領を大幅減衰。証拠駆動の自己修正
**回顧性重評估問題**: 嗅覚は回顧不要(当前感知のみ)。VeSMedはD→D推理鏈があるため、誤認領の級聯影響を遡及修正すべきか？→選項A(不回溯、衰減に任せる)が最合理。議論中

### 攻撃3: EA↔D→D反復順序（⭐⭐⭐）
**問題**: EAの結果がD→D伝播の入力になり、D→Dの結果がEAの入力になる。どちらを先に計算するかで結果が変わる。収束保証なし。
**対策候補**: 交互反復で収束を待つ（loopy BP的アプローチ）

### 攻撃4: EAが真の合併症を殺す（⭐⭐⭐⭐）
**問題**: 合併症は本物の複数疾患が共存。EAの侧抑制が強すぎると2番目の疾患を消す。弱すぎるとEAの意味がない。
**第1轮との関連**: 第1轮攻撃1と同根。soft claimingで緩和だが、softの程度の調整が新たな問題。

### 攻撃5: 計算量爆発（⭐⭐⭐）
**問題**: 430疾患×430疾患のD→D + 900変量のEA + 反復収束 = 1案例の推論に数秒〜数十秒？リアルタイム臨床使用に支障。
**対策候補**: 上位N疾患のみでEA実行（pruning）、D→Dは疎行列

### 攻撃6: 「同一回路」仮説の脆弱性（⭐⭐⭐）
**問題**: 快照内の侧抑制（競争分配）と快照間の適応（背景消去）は、概念的には「既解釈の抑制」で統一できるが、実装上は異なるデータ構造（即時計算 vs 永続記憶）が必要。「同一回路」は比喩に留まる可能性。
**対策候補**: 操作は同一（subtract explained）、データ構造は異なる、で許容

### 攻撃7: IDF削除による性能退行（⭐⭐⭐）
**問題**: IDF(dp=0.7)は現行v3.0のTop-1 78%に貢献。EAが完成する前にIDFを削除すると性能崩壊。しかしIDF+EA併用は理論的に二重計算。
**対策候補**: 段階的移行。IDF→EA置換時に性能が維持されることを確認してからIDF削除。

## EA先行研究
- **QMR-DT** (Shwe et al., 1991): 600+疾患, 4000+所見, Noisy-OR BN。EAを含むが当時計算不可能→弃用
- **Internist-1/QMR** (Myers, 1982): QMR-DTの前身、啓発式EA
- **DXplain** (Barnett, 1987): 競争的排除機制
- **Pearl** (1988): EA概念の正式定義 (*Probabilistic Reasoning in Intelligent Systems*)
- VeSMed v3.3のEAは新発明ではなく**正統BN推論への回帰** + D→D + 時間記憶
