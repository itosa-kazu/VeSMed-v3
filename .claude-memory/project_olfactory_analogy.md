## 核心発見: 三概念同一

```
嗅覚侧抑制 = Bayesian explaining away = 水流入盆地
```

全て同一操作: **強者先占、弱者分余。**

### Noisy-ORの欠陥
Noisy-ORは各疾患を独立評分。同一所見が複数疾患に同等の信用を与える（競争なし）。

### explaining awayの実装構想（v3.3）
```
1. Noisy-OR評分（現行、不変）
2. 最強疾患が最特異所見を「認領」
3. 残余疾患が未認領所見で再評分
4. 収束まで繰返し
```

出力: 排名表ではなく「疾患組合 + 各疾患が説明する所見セット」

## 核心実装決定: Soft Lateral Inhibition

8つの対抗的攻撃テスト(→ project_v33_adversarial_bio_design.md)の結果:
- **対比増強であり消灭ではない** — 視覚侧抑制はエッジを鋭化するが暗部を消さない
- **確率加重soft claiming** — hard winner-take-all禁止
- **叠加型所見(fever等)はexplaining away対象外** — 生物信号の天然属性に従う

## 推論引擎の進化路線

```
v3.0: Noisy-OR（独立評分、競争なし）
v3.2: + D→D伝播（疾患間因果）— 標準BN恢復
v3.3: + explaining away（soft lateral inhibition）
      + unexplained residual検出器（先天免疫）
      → 匹配から解釈への完全移行
```

## 形状向量仮説（NMF実験で検証済み）

### 仮説
```
嗅覚:  分子形状 × 受体形状 → 結合亲和力（物理免費計算）
医学:  疾病機制 × 変量検出特性 → CPT（???）
```

### 実験結果: 形状は存在する
- CPT矩陣(430×405)の有効次元≈74（90%分散）、嗅覚(~30)と同量級
- NMF涌現因子が自動的に器官系統/病理過程に対応
- **形状向量は実在する** — 医学知識に低秩構造がある

### 実験結果: 補完は不可能
- 交叉検証 r=0.24（予測力なし）
- Netflix反論: 類似性補完は鑑別情報を系統的に破壊する
- 残差（偏差）こそが鑑別情報 = IDFの精密版

### 根本的断裂点: 物理法則がない
- 嗅覚: 受体形状が決まれば物理が全結合亲和力を**同時・免費・精確**に決定（差異も共通性も）
- 医学: 「肺炎の機制」と「CRPの検出原理」からP(CRP↑|肺炎)=0.85を演繹する法則が**存在しない**
- → CPT値は文献手動収集が不可避

## 嗅覚類比の最終結論: 架構に有効、データに無効

| 嗅覚の要素 | 数億年データ必要? | VeSMedで実現可? |
|-----------|-----------------|----------------|
| 組合符号化 | 不要（設計原理） | ✓ 已有 |
| 侧抑制/explaining away | 不要（設計原理） | v3.3で実装可 |
| 適応/背景消除 | 不要（設計原理） | 時系列で実装可 |
| 受体選択（変量選択） | 進化で最適化だが、知的設計で代替可 | ✓ 専門家選択 |
| **物理免費CPT** | **物理法則（データではない）** | **❌ 対応物なし** |

**嗅覚がVeSMedに与える真の贈り物 = 架構蓝図（HOW）。データ捷径（WHAT）は不可。**

**Why:** 嗅覚類比の全探索。架構洞察の抽出と、データ捷径の限界の確認。
**How to apply:** 架構洞察(侧抑制/適応/組合符号化)→v3.2-v3.3で実装。CPT→大清洗(人手必須)。低秩構造→QCツール(異常CPT検出)。
