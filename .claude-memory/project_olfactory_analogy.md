## 核心発見: 三概念同一

```
嗅覚侧抑制 = Bayesian explaining away = 水流入盆地
```

全て同一操作: **強者先占、弱者分余。**

### Noisy-ORの欠陥
Noisy-ORは各疾患を独立評分。同一所見が複数疾患に同等の信用を与える（競争なし）。

### explaining awayの実装構想（v3.3）
```
1. Noisy-OR評分（現行、不変）
2. 最強疾患が最特異所見を「認領」
3. 残余疾患が未認領所見で再評分
4. 収束まで繰返し
```

## 核心実装決定: Soft Lateral Inhibition

8つの対抗的攻撃テスト(→ project_v33_adversarial_bio_design.md)の結果:
- **対比増強であり消灭ではない**
- **確率加重soft claiming** — hard winner-take-all禁止
- **叠加型所見(fever等)はexplaining away対象外**

## 推論引擎の進化路線

```
v3.0: Noisy-OR（独立評分、競争なし）
v3.2: + D→D伝播（疾患間因果）— 標準BN恢復
v3.3: + explaining away（soft lateral inhibition）
      + unexplained residual検出器（先天免疫）
      → 匹配から解釈への完全移行
```

## 形状向量仮説（NMF実験で検証済み → project_nmf_experiment.md）

### 結果要約
- CPT矩陣の有効次元≈74（嗅覚~30と同量級）、因子は器官系統に自動対応
- **形状は存在するが補完には使えない** — Netflix式補完は鑑別情報を破壊
- 嗅覚の「物理免費計算」に対応するものが医学にはない
- 残差（NMF予測からの偏差）= 鑑別情報 = IDFの精密版 → QCツールに活用可

## 浓度不変性の分析（移植不可）

### 嗅覚の浓度不変性
- 同じ気味、濃度100倍差 → 同じ気味と認識
- 機制: 侧抑制が全受体を総和で割り算（全局歸一化）
- 前提: 全400受体が**同質信号**（0-1連続値）を出力、浓度は**単一スカラー**

### VeSMedに移植不可の理由
1. **入力空間が異質** — 二値(absent/present) + 分類(macular/papular) + 連続(CRP値) + 順序(mild/moderate/high)が混在。加算・除算できない
2. **二値変量に浓度がない** — Finding層(absent/present)は大量を占めるが「浓度」概念が適用不可
3. **浓度が単一スカラーではない** — 同一疾患でも高熱+軽度咳嗽のように各症状の強度は非均一
4. **歸一化の分母が不明** — 嗅覚は400受体同時発火だが、VeSMedは900中10-20個しか観測しない。患者毎に観測子集合が異なる
5. **歸一化すべきかの循環問題** — 免疫抑制(全局衰減→歸一化すべき) vs 単なる軽症(歸一化不要)の区別に、免疫抑制の判定が先に必要

### 既存機制による部分解決
- **log-LR** = 比値形式で天然にある程度の尺度不変性あり
- **IDF** = 多疾患共有の信号を降重 → 全局性「浓度」信号を部分的に抑制

### A4(context-dependent CPT)との関係
- 浓度不変性(全局歸一化) = A4の**全局性サブセット**を自動解決する案だった
- しかし入力空間の異質性で全局歸一化が成立しない
- **残るA4案例（HIV→RPR等の局部修飾）**: 稀疏修飾層で対処（不優雅だが問題構造に正直）
- **免疫抑制**: 全局歸一化ではなくA4の稀疏修飾層で個別対処が現実的

### 結論
**概念は正しい（全局重症度が診断を支配すべきでない）が、嗅覚の具体機制は移植不可。入力空間の構造差が根本原因。log-LR+IDFが「穷人版浓度不変性」として既に機能中。**

## 嗅覚類比の最終評価表

| 嗅覚機制 | VeSMed移植可? | 理由 |
|---------|-------------|------|
| 組合符号化 | ✓ 已有 | 設計原理、データ不要 |
| 侧抑制/explaining away | ✓ v3.3 | 設計原理、データ不要 |
| 適応/背景消除 | ✓ 時系列 | 設計原理、データ不要 |
| 受体最適化(変量選択) | ✓ 専門家代替 | 知的設計が盲目進化を代替 |
| 物理免費CPT | ❌ | 医学に対応する物理法則なし |
| **浓度不変性** | **❌** | **入力空間が異質（均質信号前提が不成立）** |

**嗅覚の贈り物 = 侧抑制 + 適応 + 組合符号化。それ以外（免費CPT、浓度不変性）は入力空間の構造差で移植不可。**

**Why:** 嗅覚類比の全機制を系統的に評価。移植可/不可の境界線を確定。
**How to apply:** 移植可能な3機制(侧抑制/適応/組合符号化)に集中。浓度不変性はlog-LR+IDFで代替済み。A4はsparse modifier layerで対処。
