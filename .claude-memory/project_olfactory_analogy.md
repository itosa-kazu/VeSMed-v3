## 核心発見: 三概念同一

```
嗅覚侧抑制 = Bayesian explaining away = 水流入盆地
```

全て同一操作: **強者先占、弱者分余。**

### Noisy-ORの欠陥
Noisy-ORは各疾患を独立評分。同一所見が複数疾患に同等の信用を与える（競争なし）。

### explaining awayの実装構想（v3.3）
```
1. Noisy-OR評分（現行、不変）
2. 最強疾患が最特異所見を「認領」
3. 残余疾患が未認領所見で再評分
4. 収束まで繰返し
```

出力: 排名表ではなく「疾患組合 + 各疾患が説明する所見セット」

**Why:** 嗅覚の組合符号化から着想。VeSMedは嗅覚系統の処理鏈の大部分を既に持つが、侧抑制（explaining away）だけが欠けている。これが「匹配→解釈」の跨越の鍵。

**How to apply:** v3.3で侧抑制/explaining awayを実装。v3.0-v3.2はNoisy-OR維持。因果図の拓撲変更は不要（推論引擎の升級のみ）。

## 核心実装決定: Soft Lateral Inhibition

8つの対抗的攻撃テスト(→ project_v33_adversarial_bio_design.md)の結果、嗅覚侧抑制の正確な模倣が確定:
- **対比増強であり消灭ではない** — 視覚侧抑制はエッジを鋭化するが暗部を消さない
- **確率加重soft claiming** — hard winner-take-all禁止。生物は全層面(遺伝子→免疫→神経→進化)でsoft抑制
- **叠加型所見(fever等)はexplaining away対象外** — 生物信号の天然属性に従う

## 結合亲和力 = 病理生理形状互補（2026-04-05 新発見）

### 嗅覚の「免費データ」機制
分子有3D形状、受体有結合口袋形状。亲和力=形状互補。物理定律免費計算、不需逐対測量。

### 医学の「分子形状」= 病理生理
```
嗅覚:  分子形状 × 受体形状 → 結合亲和力
医学:  疾病機制 × 変量検出特性 → CPT
```

### CPT矩陣の低秩分解
CPT矩陣(430×900)は任意の387,000個独立値ではない。二組形状向量の内積：

```
A[430×900] ≈ D_shape[430×k] × V_shape[900×k]ᵀ
k ≈ 50-100（病理生理の有効維度）
```

維度 = 器官系統 × 病理過程 × 受累細胞 × 介質

### 嗅覚も3D空間の全精度は使わない
嗅覚の有効弁別維度 ≈ 20-40維。400受体は冗余的。精度は単個受体ではなく組合模式から来る。
→ 嗅覚~30D vs 病理生理~50-100D、量級一致

### 精度は組合から
50維 × 3精度等級 = 3^50 ≈ 7×10²³ 種可能模式。430疾患に対して十分すぎる。

### Netflix問題と同構
```
Netflix:  用戸(430) × 映画(900) × 評分(2%既知) → 予測98%
VeSMed:  疾患(430) × 変量(900) × CPT(2%既知) → 予測98%
```
NMF/SVD/autoencoderで成熟した解法あり。

### 学出維度は病理生理に対応すべし
word2vecのように、CPT矩陣分解で学出された潜在維度が自動的に病理生理概念に対応する可能性。
→ 形状向量がデータから自動涌現

### 医学教育との一致
医学生が学ぶのは二つの形状矩陣：
- 解剖+生理 = V_shape（変量の形状）
- 病理学 = D_shape（疾患の形状）
臨床推理 = D_shape · V_shape → CPT の実時計算

### 検証閉環
```
既知CPT(2%) → 矩陣分解 → k維形状向量
                        → 予測98%未知CPT
                        → 文献で予測を検証
                        → 回帰テストで診断性能を検証
```
大清洗の各CPT値 = 校正錨点 + 検証データ

**Why:** 嗅覚の「物理免費計算」をVeSMedで模倣する方法。矩陣分解=物理法則の近似。
**How to apply:** 既存8201 CPTでNMF実験 → 学出維度の解釈 → 予測CPTの文献検証。v4.x方向。
