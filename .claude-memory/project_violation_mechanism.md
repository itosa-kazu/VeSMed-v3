## 違和感メカニズムの概念

臨床専門家の「何かおかしい」という直感をBNで定量化する試み。

**定義:** 各観察済み証拠 e_i について:
```
違和感(e_i) = P(e_i | 挑戦者疾患) / P(e_i | 現Top-1)
```
比が大きい = Top-1ではうまく説明できない所見 = 違和感の源。

**手順:**
1. 全観察済み証拠について、Top-1より上手く説明できる疾患(挑戦者)を特定
2. 挑戦者ごとに違和感スコアを合算してランキング
3. 上位挑戦者に対して、KL divergenceで最も鑑別力の高い未実施検査を推奨

**利点:** 超参数ゼロ、純粋にモデルの尤度比から導出。

## PMC419768(61M骨髄腫FUO)での検証結果

### 成功した点
- D114(骨髄腫)が违和感5位として検出された
- L11=normal(肝酵素正常)がStill病の违和感(x4.2)として正しく同定
- D114の鑑別検査としてL48(SPEP), L49(骨X線), L50(TP)が正しく推奨された

### 未解決の課題
1. **挑戦者の迷走**: ブルセラ(1位)→麻疹(2位)→…と挑戦者が次々変わり、骨髄腫(5位)に辿り着かない
2. **Top-1依存**: Top-1が変わると违和感ランキングも全く変わる(D114が5位→22位)
3. **検査結果欠損**: 推奨検査(E26 Koplik斑等)の結果が原病例にない場合、詰まる
4. **単一検査限界**: L48もL50も単独ではTop-1(Still病43%)を覆せない。2検査の組み合わせが必要

### 同時に発見した構造的解決
- **L50(血清総蛋白)を追加** → TPが常規検査として入力されれば:
  - TP=very_high入力 → E21=absent確認 → **L48(SPEP)が推奨1位に浮上** → monoclonal → D114 Top-1(81%)
  - 超参数なし、モデル構造だけで3ステップ到達
- ただしTPは「システムが推奨する」のではなく「常規検査として自動的に手元にある」前提

### 突破口：エントロピー増大方向の検査推奨（反証推奨）

尤度比ベースの違和感メカニズムは挑戦者が迷走する問題があったが、
**エントロピー増大方向の検査推奨**で解決：

**定義:** 各未観測検査vの各状態sについてH_after(v=s)を計算。
H_after > H_now となる状態がある検査 = 現Top-1を動揺させる検査。
max_s(H_after)が最大の検査を反証推奨1位とする。

**検証結果(PMC419768 骨髄腫):**
- 反証推奨: **5ステップ**でD114 Top-1到達(5.7%)、その後L48/L50で98%
- 通常推奨: 18ステップでも到達せず(D114はrank16-19に沈没)
- 正診時(R01 Still病): 反証推奨の実結果がTop-1を補強(30%→99.9%)

**通常推奨 vs 反証推奨:**
- 通常推奨(IG): 確認バイアスあり。正しければ速いが、間違うと修正不能
- 反証推奨: 一時的に不安定だが自己修正可能。誤診時に修正、正診時に補強

**実装方針:** 反証推奨を主、通常推奨を副として2枠並列表示。
- 超参数ゼロ、純粋に情報論ベース
- 計算コスト: next_best_testと同じ(H_afterは既に計算済み)
