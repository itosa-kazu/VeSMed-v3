---
name: 因果関係疾患の重複問題
description: D273(総胆管結石)とD25(急性胆管炎)のように因果関係にある疾患が鑑別として競合し確率を食い合う問題
type: project
---

D273(総胆管結石)はD25(急性胆管炎)の最も一般的な原因(50-70%)。鑑別診断ではなく因果関係。

**問題**: 同じevidence pattern(黄疸/胆管拡張/胆汁うっ滞/直接Bil優位)を共有し、確率を食い合う。テストケースでD25=80%+D273=7%=87%で実質同じ病態を指している。

**推奨解決策(方案2)**: D273を「無症候性/非感染性胆管結石」に再定義し、D25を「急性胆管炎(原因問わず)」とする。発熱+胆管感染があればD25、なければD273。

**Why:** 因果関係にある疾患を独立した鑑別として扱うと、確率分散で両方のrankが下がる。
**How to apply:** 将来的にD273のevidence patternを非感染性に限定し、感染徴候(発熱/WBC/CRP)がある場合はD25に集約する。同様のパターンが他にもないか確認（例: 胆嚢結石→胆嚢炎）。
