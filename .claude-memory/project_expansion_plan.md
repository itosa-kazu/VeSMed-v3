## 方針
システム架構は疾病無関。BN推論エンジンはそのまま使える。疾患・変量・辺を追加するだけで拡張可能。

## 拡張ロードマップ

### Phase 1: 呼吸困難鑑別 ← 完了
追加済み(D120-D130): 心不全, COPD増悪, 喘息発作, 気胸, タンポナーデ, 不整脈, アナフィラキシー, 胸水, 上気道閉塞, 過換気, ILD
新変量: L51 BNP, S49起座呼吸, E36下腿浮腫, E37 JVD, R44心疾患既往, R45喫煙歴, R46 COPD既往, R47喘息既往
E07+decreased_absent, L04+pneumothorax
残り: D124タンポナーデの案例(3件), ARDS(症候群→保留)

### Phase 2: 胸痛鑑別 ← 次
ACS(急性冠症候群), 大動脈解離, GERD, 肋骨骨折, 肋間神経痛 等
PE/心筋炎/心膜炎/気胸は既存

### Phase 3: 腹痛鑑別
大部分が既存(D10虫垂/D11胆嚢/D86膵炎等)。追加は少ない

### Phase 4: 意識障害鑑別
脳卒中, 低血糖, 薬物中毒, てんかん重積 等
髄膜炎/脳膿瘍/脳炎は既存

### Phase 5: 全部統合
**Prior調整が必要**: 現在のpriorは発熱門診の患者集団を前提(BASE=0.01)。
全疾患に拡張する場合、一般救急外来の有病率分布に合わせて再calibrationが必要。

## マスター疾患リスト
`疾患リスト.txt` に約670疾患の完整清单あり（国試出題基準+研修医必修）。
我们の137疾患との差分分析で系統的に漏れを特定可能。
高優先未追加: DKA/低血糖/脳梗塞/脳出血/てんかん重積/GBS/誤嚥性肺炎/尿管結石等

## 不漏のための系統的アプローチ
1. 教科書/ガイドラインの鑑別リストを参照
2. ER見逃し率Top-15等のエビデンスベースリスト
3. 臓器系統別に網羅確認
4. 追加後に既存案例の回帰テスト必須
