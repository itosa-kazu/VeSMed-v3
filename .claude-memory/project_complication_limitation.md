# 合併症表現の構造的限界

現在のVeSMedモデルは「疾患→所見」の1層構造であり、「疾患A→合併症B→所見」のような因果連鎖を直接表現できない。

**具体例:**
- 肝硬変→門脈圧亢進→食道静脈瘤→吐血
- SLE→ループス腎炎→ネフローゼ
- 糖尿病→DKA→意識障害

**現状の対処法:**
- 中間変量(L64 DIC, L65 心不全, L66 肝障害, L67 AKI)で部分的に対応
- しかし「合併症」自体を疾患ノードとして扱う仕組みはない

**Why:** BNの構造的制約。疾患ノード間の因果関係（D→D）はモデルに存在しない。

**How to apply:** **→ 万有医学因果図のD→D辺で解決（v3.2）。** hierarchical BN不要。D→D辺を追加するだけで合併症は図上の隣接ノードとして自然表現される。
