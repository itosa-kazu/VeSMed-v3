**新変量追加後は、全疾患リストと手動照合して辺の漏れを確認する。**

**Why:** S59(めまい)追加時に7疾患のみに辺を張ったが、手動照合で不整脈/CO中毒/ウェルニッケ/減圧症/TIA/もやもや/NPH等7疾患の漏れを発見。S60(頭痛パターン)でも日本脳炎/HSV脳炎/結核性髄膜炎/脳膿瘍/EDH/SDH/水頭症等10疾患が漏れていた。

**Parent数が少ない変量はIDF≈1.0で特定疾患を偏袒する**（feedback_trinity_audit項目8）ので、網羅的に辺を張ることが重要。

**How to apply:**
1. 新変量追加後、`step1_fever_v2.7.json`の全疾患リストを表示
2. 各疾患のnoteを読み、その疾患がこの変量の値を**直接引き起こすか**を判断
3. YESなら辺+CPTを追加
4. 最終的にparent数が適切(≧3)になるまで繰り返す
5. 三位一体確認

**Skill(/add-diseases)にも組み込むべき:** 新変量追加ステップに「全疾患照合」を必須チェックとして追加。
