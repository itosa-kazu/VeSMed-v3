推奨検査機能の改善：検査名だけでなく「期待所見」も同時表示する。
つまり、その検査でどの結果が出たらエントロピーが最も減るかを表示。

**Why:** 臨床医にとって「何を検査すべきか」だけでなく「何を期待して検査するか」が重要
**How to apply:** next_best_test()の結果に、各検査の各state別エントロピー減少量を追加し、最大減少stateを「期待所見」として表示
