## 中間変数化の実験と教訓（2026-03-14）
E03直接辺をM02経由に変更する実験を実施 → Top-3 146→144に退化。

**根本原因**: log-LR辺は「正の信号」と「負の信号（反証）」の両方を運ぶ。
- E03=低血圧 → ショック疾患にboost（正、M02で代替可能）
- E03=正常 → ショック疾患にpenalty（負、M02では代替不可能）
辺を削除すると反証情報も失われ、ショック疾患がE03=正常でもpenaltyを受けなくなる。

**結論**: M02は「並列sign変数」として維持。中間変数化は不採用。
M01(SAH)が成功したのはSAH固有症状（頸部硬直等）が他疾患では出ないため。
E03(血圧)のような全身変数は直接辺を残す方が情報量が多い。
