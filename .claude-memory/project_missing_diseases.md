## 追加済み ✅
- D105 パルボB19, D106 DM/PM, D107 HLH, D108 B型肝炎, D109 クリプトコッカス
- D110 トキソプラズマ, D111 SJS/TEN, D112 PAN, D113 ヒストプラズマ, D114 骨髄腫
- D115 播種性淋菌感染症(DGI), D116 急性心筋炎

## 低優先（残り）
- **混合性結合組織病(MCTD)** — SLE/DM/SSc重複
- **骨髄異形成症候群(MDS)** — 高齢者の汎血球減少
- **ツラレミア(野兎病)** — 動物接触+リンパ節腫脹
- **バベシア症** — ダニ媒介、マラリア類似

## 高優先（ER見逃し率Top-15から）
- **脊椎硬膜外膿瘍** — 見逃し率56%(全疾患最高)。発熱+背部痛+神経症状。初期は背部痛のみで見逃される
- **脳膿瘍** — 発熱+頭痛+局所神経症状。髄膜炎/脳炎と誤診されやすい

## 新候補
- **コクシジオイデス症** — 渡航者の肺炎+結節性紅斑
- **ノカルジア症** — 免疫不全+肺+脳膿瘍
- **腎周囲膿瘍** — 腎盂腎炎の重症鑑別
- **リステリア髄膜炎** — 高齢者/免疫不全
- **感染性大動脈瘤** — 稀だがFUO原因
