## 追加済み ✅
- D105-D109: パルボB19, DM/PM, HLH, B型肝炎, クリプトコッカス
- D110-D114: トキソプラズマ, SJS/TEN, PAN, ヒストプラズマ, 骨髄腫
- D115-D118: DGI, 急性心筋炎, 脊椎硬膜外膿瘍, 脳膿瘍

## 低優先（残り）
- **混合性結合組織病(MCTD)** — SLE/DM/SSc重複
- **骨髄異形成症候群(MDS)** — 高齢者の汎血球減少
- **ツラレミア(野兎病)** — 動物接触+リンパ節腫脹
- **バベシア症** — ダニ媒介、マラリア類似

## 新候補
- **コクシジオイデス症** — 渡航者の肺炎+結節性紅斑
- **ノカルジア症** — 免疫不全+肺+脳膿瘍
- **腎周囲膿瘍** — 腎盂腎炎の重症鑑別
- **リステリア髄膜炎** — 高齢者/免疫不全
- **感染性大動脈瘤** — 稀だがFUO原因
