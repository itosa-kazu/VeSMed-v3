**原子化原則: 異なる検体・手技・感度を持つ検査は別変量にする。**

**Why:** 喀痰Gram染色とBAL鏡検を1変量に混合しようとした際に発見。同じ形態結果(例:GPC陽性)でも、喀痰は口腔常在菌混入で偽陽性リスクが高く、BALは無菌採取で診断的意義が格段に高い。混合すると「GPC on BAL」と「GPC on sputum」の診断重みの差がモデルに反映できない。

**How to apply:**
- 新変量設計時に「この変量名に2つ以上の検体/手技が含まれていないか？」を確認
- 判断基準: 同じ結果でも**診断的意義が異なる**なら別変量
- 例: 喀痰Gram(L105) と BAL鏡検(L106) は別。血培Gram と 喀痰Gram は別
- 例外: CXR(PA/側面/portable)のような同一検査の撮影条件差は統一OK
