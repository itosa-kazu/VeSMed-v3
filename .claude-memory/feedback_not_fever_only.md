VeSMedは全疾患を対象とする包括的システムである。「発熱domain」に限定して考えてはならない。

**Why:** ユーザーから明確に指摘された。「为什么你总之纠结于发热疾病？我们先做的系统是包罗所有病的。」発熱は出発点に過ぎず、既にACS/大動脈解離/脳卒中/DKA等の非発熱疾患も多数含まれている。

**How to apply:**
- 新疾患追加時に「発熱domainではscope外」と判断しない
- 全科の重要疾患を対象に追加を検討する
- 疾患の追加可否は「臨床的重要性」で判断する（発熱の有無ではない）
