---
name: CPT補0禁止
description: 欠落stateに0%を機械的に補填するとLR虚高になる。文献ベースで適切な値を設定すること
type: feedback
---

CPT state不一致修正時、欠落stateに0.0%を一律補填した結果、max|logLR|が虚高になる問題が発生。

例: D76(輸血反応) S07(倦怠感) severe=0% → LR=54.6x（実際は5%程度が妥当）

**Why:** P(state)=0だとLR=∞になり、max|logLR|が4.0(cap値)まで跳ね上がる。臨床的にはゼロであることは稀で、1-5%が妥当な場合が多い。

**How to apply:** 欠落stateを補填する場合は:
1. 文献(PMC/教科書)でその疾患のその所見の頻度を必ず検索
2. デフォルト値禁止。毎回文献ベースで正確な値を設定
3. 補填後は合計が1.0になるよう他のstateから按分減算
4. 精修フェーズ: 問題のある全CPTを1つずつ文献検索して手動修正
