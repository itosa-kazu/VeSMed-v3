新辺（D→変量）の追加は、LLMが「この疾患はこの症状を引き起こすはず」と思うだけでは不十分。**PMC/教科書等の文献根拠が必要**。

**Why:** LLMの医学知識は確率的であり、誤った因果関係を追加すると過拟合や性能低下を招く。CLAUDE.mdの「CPT変更は文献必須」ルールは新辺にも同様に適用される。

**How to apply:**
1. 辺追加前にWebSearchでPMC/教科書を確認
2. 各辺に`reason`フィールドで文献根拠を記載
3. LLMの「印象」だけで辺を追加しない
4. R→D prior（年齢/性別の疫学データ）は公知の疫学事実なので比較的許容されるが、D→症状/検査の因果辺は臨床エビデンスが必要
