---
name: 臨床正確な辺の撤回禁止
description: テスト退化を理由に臨床的に正しい辺を撤回するのは過拟合。正解は競合疾患にも同じ辺を追加する公平性
type: feedback
---

テスト結果が悪化したからといって、臨床的に正しい辺を撤回してはならない。それはテストデータへの過拟合。

**Why:** D222(RA)→S46(食欲不振)を追加したらR445(シェーグレン)が退化。辺を撤回したらユーザーに「作弊」と指摘された。RAの食欲不振は臨床的事実。

**How to apply:**
1. 退化が発生したら、まず**競合疾患にも同じ辺があるか確認**
2. 競合疾患も同じ臨床所見を持つなら→そちらにも辺を追加（公平性）
3. CPT値が不適切なら→値を調整
4. それでも退化するなら→**受け入れる**（臨床的に難しい鑑別は正直に認める）
5. **絶対に「テスト結果のために正しい辺を消す」ことはしない**
