## 核心発見: 侧抑制と適応は同一操作

嗅球の顆粒細胞(granule cell)は**2つの役割を同時に**担う:
1. **快照内**: 侧抑制（異なる気味成分間の競争）
2. **快照間**: 適応（前回の嗅探の記憶を減算）

```
僧帽細胞出力 = 新入力 − 顆粒細胞抑制
             = 新入力 − 旧状態の記憶
             = 変化量（差分）
```

**差分計算は処理回路に内建。別層不要。**

### 同一操作の2つの作用範囲
- 快照内「他の疾患に説明済み」= 侧抑制
- 快照間「前回既に説明済み」= 適応
- **抑制対象が違うだけで操作は同じ: 既説明の抑制**

## 信息=変化（核心原理）

嗅覚系統はdX/dt(一階導数)を符号化、X(絶対値)ではない。
Shannon信息量=意外程度=偏離預期。一直存在の症状は情報を運ばない、変化こそが情報。
状態記憶付きexplaining awayは自動的にdX/dtを計算。

## VeSMed映射: explaining away + 状態記憶

### 方案比較

| | 方案B(独立快照+減法) | 方案C(快照+時間層) | **嗅覚式(EA+記憶)** |
|--|------|------|------|
| 推論引擎 | 不変 | 不変 | explaining away(v3.3) |
| 時間処理 | 手動比較 | 単独後処理層 | **推論回路に内建** |
| 記憶 | なし | 外部 | **状態記憶(顆粒細胞)** |
| 生物対応 | なし | 人工 | **嗅球の精確復刻** |

### 具体動作

```
t0: 患者入院
    evidence = {fever, cough, infiltrate}
    explaining away → 肺炎が認領{fever, cough, infiltrate}
    状態記憶: {fever→肺炎, cough→肺炎, infiltrate→肺炎}

t1: 2日後、新増浮腫+BNP↑
    evidence = {fever, cough, infiltrate, edema, BNP↑}
    状態記憶: fever/cough/infiltrate は肺炎で説明済み
    → {edema, BNP↑}のみ未説明 → explaining awayで処理 → 心不全
    出力: 肺炎(fever,cough,infiltrate) + 心不全(edema,BNP↑)

t2: 抗生素後、熱退+CRP正常化
    肺炎の認領症状が消退 → 肺炎緩解
    心不全の症状は不変 → 心不全持続
```

**合併症が時間で自然分離。後処理不要。**

## 濃度不変性 = 模式比例識別

嗅覚系統は受体激活の**比例関係**(2:6:1)を識別、絶対値ではない。
機制: 侧抑制が絶対強度を帰一化、相対対比度のみ残す。

VeSMed映射: "濃度"=症状重症度。免疫抑制=信号産生能力低下="濃度"降低。
現在VeSMedは濃度に敏感(CRP=mild vs high走不同CPT)→上下文変量問題に帰着。

## 根本結構差異: 嗅覚系統に疾患節点なし

- **変量 ≈ 受体** — 大致対応
- **CPT ≈ 結合親和力** — 但辺とCPTは嗅覚で同一物理実体
- **辺** — 独立概念として不存在、CPTと融合
- **疾患節点** — **完全不存在**。嗅球に"咖啡"節点なし。咖啡=400受体の激活模式本身

VeSMed=二部図(430疾患↔900変量)。嗅覚=模式空間(識別=最近傍匹配)。
**VeSMedの"疾患は顕式節点"という仮設は嗅覚系統で不成立。模式=診断。**
