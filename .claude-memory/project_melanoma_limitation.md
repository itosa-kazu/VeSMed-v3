D423悪性黒色腫のテスト表現が極めて悪い: Top-1 2/12(17%), Top-3 5/12(42%)。

**Why:** モデルに皮膚病変を記述する変量がない。黒色腫の最も鑑別力のある所見（色素性病変、ABCDE基準、爪下色素沈着等）が表現不可。転移性黒色腫の全身症状（体重減少/LDH上昇/転移画像）は胃癌/SCLC/前立腺癌等と重複度が高い。

**How to apply:**
- 皮膚所見変量の追加を検討（例: S_skin_lesion: absent/pigmented_irregular/amelanotic_nodule/subungual_pigmentation）
- 追加すれば黒色腫だけでなく、基底細胞癌/扁平上皮癌/乾癬(D410)/丹毒(D411)等にも辺を張れる
- 追加までは、D423の低Top-1は受容する（辺追加では解決不可能）
- R1162/R1143: p_obs=0 bug（女性に前立腺癌rank 1）も寄与
