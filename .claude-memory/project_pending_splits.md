## 分割タスク

### AML/ALL分割（D68 急性白血病）— 最優先
- D68を AML + ALL に分割
- AML: 中高年(median 68), DIC/出血多い, Auer小体, 歯肉腫脹(M4/M5)
- ALL: 小児(2-5歳)+高齢, 肝脾腫/LAD多い, TdT+, CNS浸潤
- **Why:** 年齢分布・臨床像・合併症が全く異なる。umbrella化で鑑別力低下

## 新疾患追加タスク

### ネフローゼ症候群（MCD + MN + FSGS）— 3疾患
- 微小変化型(MCD): 小児(2-6歳), 突然発症, ステロイド著効, 血尿少ない
- 膜性腎症(MN): 成人(50-60歳), 潜行性, PLA2R抗体, ステロイド不応
- FSGS: 任意年齢, アフリカ系多い, 高血圧+血尿, ステロイド不応
- **Why:** 年齢・発症速度・血尿・高血圧で生検前にある程度鑑別可能
