## D→Dは嗅覚に対応なし — GRNが対応

嗅覚系統に疾患ノードなし→D→D辺の対応物なし。
基因調控網絡(GRN)が正確な対応: 基因→基因の調控 = D→D辺。

## GRNから抽出したD→D設計規則

| GRN原則 | VeSMed D→D設計 |
|---------|---------------|
| 局部伝播 | 一歩因果のみ、多歩効果は鏈式涌現 |
| 信号衰減 | 自然衰減（各歩の伝播確率の積） |
| **阈値(Hill函数)** | **explaining awayから涌現（パラメータ不要）** |
| 稀疏網絡 | 各疾患2-5条D→D辺、総計~1000-2000条 |
| 網絡母題 | 前馈環は自然出現 |
| Master regulators | 免疫抑制/DM/肝硬変=hub、優先収集 |
| 表観遺伝 | D→D辺もA4 modifier影響下 |

## 阈値はexplaining awayから涌現

### GRNの阈値機制
- Kd(解離定数) = 阈値位置。分子形状互補性から決定。**辺固有**属性
- Hill係数n = 阈値陡度。協同結合から決定。**辺固有**属性
- 阈値はパラメータではなく物理属性の結果

### VeSMedの阈値: explaining away = Hill函数
```
GRN:    協同結合(n>1) → sigmoid → 二値化
嗅球:   侧抑制 → 赢者通吃(soft) → 二値化
VeSMed: explaining away → 競争的分配 → posterior鋭化 → 二値化
```
三者は同一の数学操作。explaining awayが天然に阈値を生む。

### 推論順序
```
1. Noisy-OR原始評分
2. Explaining away → posterior鋭化（≈Hill函数）
3. D→D伝播（鋭化後のposteriorを使用、追加阈値不要）
4. 再度explaining away → 収束
```
**阈値はパラメータではなく、explaining awayの涌現属性。ゼロ新パラメータ。**

## D→D辺強度: 精度は実験で決定

### GRNの辺強度は連続的・精確（三档ではない ← 修正）
GRNのKd値は数桁にわたる連続値。進化が数十億年かけて各辺を精確に調整。
「三档で十分」は誤り。GRNは精確だが、それは数十億年のデータがあるから。

### VeSMedのD→D辺強度: 段階的精密化
D→D辺はD→E辺と異なり**開門（候補リスト入り）**であり**定位（最終順位決定）**ではない。
最終順位はD→E証拠とexplaining awayが決定。D→D辺強度の精度がどこまで必要かは不明。

### 実験方案
```
A. 拓撲only（辺存在=固定boost）← 最簡
B. 三档（strong/moderate/weak）
C. 連続精確CPT（辺ごとに確率値）← 最精確
→ 各方案で回帰テスト、合併症案例でTop-1/Top-3を比較
```

**原則: 最簡から始め、実験が要求する精度まで段階的に上げる。**
GRNは進化(=実験)で精度を上げた。VeSMedも回帰テスト(=実験)で精度を上げる。

