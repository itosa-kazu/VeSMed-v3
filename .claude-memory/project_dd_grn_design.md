## D→Dは嗅覚に対応なし — GRNが対応

嗅覚系統に疾患ノードなし→D→D辺の対応物なし。
基因調控網絡(GRN)が正確な対応: 基因→基因の調控 = D→D辺。

## GRNから抽出したD→D設計規則

| GRN原則 | VeSMed D→D設計 |
|---------|---------------|
| 局部伝播 | 一歩因果のみ、多歩効果は鏈式涌現 |
| 信号衰減 | 自然衰減（各歩の伝播確率の積） |
| **阈値伝播(Hill函数)** | **sigmoid門控: P(上流)<阈値時に不伝播** |
| 稀疏網絡 | 各疾患2-5条D→D辺、総計~1000-2000条 |
| 網絡母題 | 前馈環(糖尿病→感染+腎病)は自然出現 |
| Master regulators | 免疫抑制/DM/肝硬変=hub、優先収集 |
| 表観遺伝 | D→D辺もA4 modifier影響下 |

## 阈値伝播の詳細

### なぜGRNは阈値を発達させたか
生物信号は本質的にノイズだらけ（確率的転写、分子揺らぎ）。
線形伝播だと鏈の各歩でノイズが蓄積→数歩後に信号がノイズに埋没。
阈値(sigmoid)=各歩でノイズ濾波+信号再生→デジタル電子回路と同原理。

### VeSMed対応
- P(胆管結石)=0.1 → 線形伝播: 胆管炎に0.1×w boost（ノイズ）
- P(胆管結石)=0.9 → 線形伝播: 胆管炎に0.9×w boost（信号）
- 阈値伝播: P<阈値なら伝播しない → 不確実仮説が下流を汚染しない

### 実装案
```python
# 線形（天真版）:
boost = w * P(upstream)  # 全仮説が伝播、ノイズ蓄積

# 阈値（GRN式）:
boost = w * sigmoid(P(upstream) - threshold)  # 確信度高い時のみ伝播
```
